# Проект 3: Операционный BI-отчёт в Power BI

Финальный проект — полноценный операционный отчёт для логистической компании.
Анализируем выручку, прибыль, задержки по регионам и менеджерам.

## Что внутри

- KPI-карточки: заказы, выручка, затраты, процент задержек
- Динамика выручки по месяцам
- Прибыль по регионам
- Задержки по перевозчикам
- Выручка по категориям товаров
- Два среза: по региону и менеджеру

## Новое по сравнению с Проектом 2

- KPI-карточки для ключевых метрик
- Линейный график динамики
- Меры с зависимостями: Прибыль = Выручка - Затраты
- Два среза одновременно

## Инструменты

Power BI Desktop, DAX, Power Query, CSV
